Open up Illustrator

Create a 5in x 5in document.

Using the line tool create a 1 inch vertical black line with a 1pt stroke.

Then create a 1 inch horizontal black line with a 1pt stroke. Make this line connect at the bottom of the vertical line to create an ‘L’ shape.

Create two more vertical black lines with 1pt stroke.

The top of one line will connect to the end of the horizontal line.

The top of the second line will be parallel to the other but sit .16in to the left.

Lastly create one more horizontal black line.

This line will sit .13in below the first horizontal line.

The start of the line should connect to the vertical line that sits .16in away from the other.
